💡 Rol Kullanım Örnekleri

Playbook'ta Rol Kullanımı:

---
- name: Web Sunucularını Yapılandır
  hosts: webservers
  become: yes
  roles:
    - common
    - webserver

- name: DB Yapılandır
  hosts: databases
  become: yes
  roles:
    - common
    - database

Rol Değişkenleri:

- name: Rol çalıştır (değişken geçir)
  hosts: all
  roles:
    - role: webserver
      vars:
        http_port: 8080
        server_name: prod
