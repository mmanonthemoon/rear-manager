📖 Format Referansı

Linux Satır Formatı:

name, hostname, linux, port, user, şifre, become, grp, notlar

Sütun  Açıklama               Örnek
──────────────────────────────────────────
1      Ad (zorunlu)            web01
2      Hostname/IP (zorunlu)   192.168.1.10
3      OS tipi (opsiyonel)     linux
4      SSH port                22
5      SSH kullanıcı           ubuntu
6      SSH şifre               P@ss
7      Become                  sudo / su / none
8      Become kullanıcı        root
9      Become şifre aynı mı    1 (evet) / 0 (hayır)
10     Grup adı                linux_prod
11     Notlar                  Web sunucusu

Windows Satır Formatı:

name, hostname, windows, port, user, şifre, transport, domain, grp, notlar

Sütun  Açıklama               Örnek
──────────────────────────────────────────
1      Ad (zorunlu)            win-app01
2      Hostname/IP (zorunlu)   192.168.1.100
3      OS tipi (opsiyonel)     windows
4      WinRM port              5985
5      Kullanıcı               Administrator
6      Şifre                   WinP@ss
7      Transport               ntlm / basic / kerberos
8      Domain                  CORP.LOCAL
9      Grup adı                windows_domain
10     Notlar                  DC Sunucu

Örnek CSV İçeriği:

Şablonu indir:

✓ Grup adı mevcut bir gruba eşleşmezse atanmaz (hata olmaz).
✓ Aynı isimde host zaten varsa satır atlanır, hata gösterilir.
✓ os_type sütunu atlanırsa "Varsayılan OS Tipi" kullanılır.
